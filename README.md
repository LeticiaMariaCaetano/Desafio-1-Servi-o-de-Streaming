# Desafio-1-Serviço-de-Streaming
Modelagem e criação de um pequeno grafo de conhecimento para um serviço de streaming. O modelo contém:   **Entidades (Nós):** User, Movie, Series, Genre, Actor, Director  **Conexões (Relacionamentos):** WATCHED (com propriedade rating), ACTED_IN, DIRECTED, IN_GENRE.
